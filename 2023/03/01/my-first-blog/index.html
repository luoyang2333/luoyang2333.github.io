这是我的第一篇博客

我的博客搭建是在CSDN网站上学习的,这篇文章的地址是：

零基础搭建博客

这篇文章前部分讲述如何快速搭建博客，后部分讲述将博客设置自己的快速访问域名

接下来主要来回顾一下搭建博客的基本步骤：

1.首先需要搭建环境

  • 开发工具 webstorm

    webstorm

  • node.js 环境

    node.js

  • git 环境

    git

  • hexo 框架的搭建 (安装完git后)

    在桌面右键单击，选择git bash here，输入 npm install -g hexo-cli 等待。
    对hexo进行初始化，并进行环境配置。

2.hexo的初始化

  • 在除C盘之外的任意盘新建文件夹，命名为自己的博客，作为博客的文件夹
  • 打开此博客文件夹，右键单击，选择 git bash here,输入：

    hexo init

  • 如果报错，可能是网络原因，成功之后，会在博客目录新建许多文件

3.用 webstorm 打开博客文件夹（直接将文件夹拖进 webstorm）

  • 这时可以打开终端 terminal
  • 输入：

    hexo s

  • 这时会返回一串网址，点击直接打开，我们会看到hexo内置的网页，接下来就需要选择我们博客的主题了。

4.安装主题

  • 大家可以打开官方主题库选择自己心仪的主题

    官方主题

  • 应用心仪的主题，我这里选择butterfly，在 terminal 输入：

    git clone -b master https://gitee.com/immyw/hexo-theme-butterfly.git themes/butterfly

  • 等待安装完成，安装成功后会在博客文件夹下的 themes 生成 butterfly 的文件夹

  • 安装渲染插件 pug 与 stylus ，控制台依次输入：

    npm install hexo-renderer-pug hexo-renderer-stylus –save

  • 还有一个忘了在哪篇文章来着，不管这个，先运行第一个，然后再在terminal输入：

    • hexo clean
    • hexo g
    • hexo s

  • 之后会返回一个地址，点开，若网页出现报错，复制这个报错搜索一下就能找到解决方法。
    这时的博客很简单，需要对其进行配置，我们放到后面去讲。

5.github pages 配置

  • 在完成了前面的博客的初步搭建，我们需要将博客上传到一个域名上，这样，就能通过访问这个域名直接到达我们到达我们自己的博客。

  • 首先需要注册 github 账号

    github主页

  • 注册之后需要新建一个仓库

  • 一定要选择 public ，名称一定要是用户名，然后创建

  • 如果是第一次在你的电脑上使用git，则需要配置SSH公钥，类似于验证码之类的。操作如下：

    • 在桌面右键单击，选择 git bash here
    • 依次输入 git config –global user.name ‘用户名’

      git config –global user.email ‘邮箱’ 注：这里的用户名和邮箱与 github 保持一致。
    • 输入 ssh-keygen -t rsa -C ‘github邮箱’ ，连按回车到结束，这时在返回值里边可以找到一个本地路径，我们按照这个路径找到 id_res.pub 文件。
    • 用记事本打开这个文件，并全选复制
    • 打开 github 网站，单击头像，选择settings，在左侧栏中选择SSH and GPG keys
    • 在SSH keys 处选择new SSH key
    • 把复制的内容粘贴，自己去一个 title，点击 add SSH key 即可。

  • SSH 配置完成之后，打开 webstorm 在菜单栏点击 git ，选择 manage remotes ，点击加号添加github地址

6.修改网站部署

  • 打开 webstorm ，在左侧栏中选择_config.yml ,双击打开
  • 在最后面有一个 deploy，按图中修改参数
  • 在博客根目录下右键单击，选择git bash here，依次输入：

    hexo clean

    hexo g

    hexo s ， 注：这里会生成一个网址，这就是你的博客预览，按Ctrl+C结束。

    hexo d

  • 执行完毕后，就可以通过“你的用户名+.github.io”访问你的博客网站了！

7.域名的注册

  • 在成功完成上述操作后，我们仅仅能够通过 github.io 来访问我们的博客，而且由于github服务器在国外，所以有时候访问会比较慢，因此，我们需要一个域名来快速访问。
  • 域名的注册主要有以下几种方法：

    1.通过申请 github 学生包来白嫖域名;
    注：网上有很多的教程，这里就不介绍，值得注意的是现在难申请。

    2.申请国内的阿里云或者腾讯云，自己百度进入官网申请即可。

  • 在注册好域名之后，需要对域名进行解析。因为我用的是 github 学生包申请的域名，我并没有服务器，因此这里就推荐将域名托管到 cloudflare，这个网站提供免费的域名解析。

    cloudflare
